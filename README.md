# Quartet Concert Website – Backend

Django REST Framework backend for a concert/quartet website.  
Provides API endpoints and authentication for managing concert data.

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite/PostgreSQL

## Features
- REST API for concert data
- User authentication system
- Create and manage concerts
- Django admin panel for backend management
- Full integration with React frontend

## How to run locally

pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver  

You can access the Django admin panel to create users and manage concerts, or use the frontend login system.
